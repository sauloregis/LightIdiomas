#Light Idiomas - Sistema de Cadastro de Alunos
##Este projeto é um sistema de gerenciamento interno desenvolvido para a Light Idiomas. O objetivo principal é centralizar as informações de aulas, turmas e pagamentos, facilitando o acesso e a organização dos dados dos alunos.

#O sistema atualmente possui as seguntes funcionalidades:
## Dashboard Principal: Uma visão resumida com métricas dos alunos matriculados e status geral do sistema.
## Gestão de Alunos: Tela dedicada para listar e cadastrar novos alunos, incluindo informações de contato e financeiro.
##Gestão de Turmas: Espaço para organizar os níveis de ensino e horários oferecidos pela escola.
##CRUD Completo: Na tela inicial, é possível visualizar, editar ou remover cadastros de alunos de forma rápida.
